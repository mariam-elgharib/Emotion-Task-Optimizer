# Emotion-Task-Optimizer
This application combines computer vision with optimization algorithms to recommend personalized tasks based on your current emotional state. It features a modern GUI with live camera feed, task management, and an interactive session timer.

# Installation
1. Run These commands in the terminal of Visual Studio
   - python -m venv emotion_env
   - .\emotion_env\Scripts\Activate
   - pip install opencv-python
   - pip install fer
   - pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu
   - pip install numpy
2. Add Source Code
3. Run app.py
